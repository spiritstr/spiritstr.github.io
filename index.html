O que vamos fazer: Fazer update da distro, instalar os drivers de video, instalar as dependencias para o jogo.

Primeiro vamos fazer o update do sistema: 
  sudo apt update

Isso não vai ser realmente nescessario para o Starcraft 2 já que ele roda com DX9, mas pode ser util se você instalar o Battle.net com DX11 e rodar com DXVK, isso vai instalar o suporte para Vulkan.

sudo apt install libvulkan1 libvulkan-dev vulkan-utils

Se você estiver usando AMD instalte os drivers Mesa primeiro.

sudo add-apt-repository ppa:oibaf/graphics-drivers
sudo apt update
sudo apt upgrade
sudo apt install mesa-vulkan-drivers

Se estiver usando Nvidia instale os drivers.

  sudo add-apt-repository ppa:graphics-drivers/ppa
  sudo apt update
  sudo apt upgrade
Instalando os drivers
A duas maneiras de instalar, via GUI ou terminal.
GUI

Você só precisa abrir a Configuração do Sistema,então Programas e Atualizações, Drivers Adicionais e clicar em aplicar, isso pode levar alguns minutos então espere.
Terminal

Para instalar via terminal você precisa rodar o seguinte comando, para descobrir qual driver está disponivel. 
ubuntu-drivers devices
Ele vai retornar linhas simulares a essas com os drivers recomendados: 
vendor   : NVIDIA Corporation
model    : GP104 [GeForce GTX 1070]
driver   : nvidia-driver-390 - third-party free
driver   : nvidia-driver-396 - third-party free recommended
driver   : xserver-xorg-video-nouveau - distro free builtin
Rodar a seguinte linha vai instalar automaticamente o ultimo driver. 
sudo ubuntu-drivers autoinstall
Você também pode instalar um driver em particular como o exemplo: 
sudo apt install nvidia-396
Instalando o Wine
A duas maneiras de fazer isso, via GUI e via Terminal.
Para sistemas 64bit é preciso habilitar a aquitetura de 32bit. 
sudo dpkg --add-architecture i386
wget -nc https://dl.winehq.org/wine-builds/Release.key
sudo apt-key add Release.key
sudo apt-add-repository https://dl.winehq.org/wine-builds/ubuntu/

Now we should install Wine, i recommend staging over the stable branch.

Stable:
sudo apt-get install --install-recommends winehq-stable
staging:
sudo apt-get install --install-recommends winehq-staging
Instalando Lutris
Rode as seguintes linhas uma por uma. 
ver=$(lsb_release -sr); if [ $ver != "18.04" -a $ver != "16.04" ]; then ver=18.04; fi
echo "deb http://download.opensuse.org/repositories/home:/strycore/xUbuntu_$ver/ ./" | sudo tee /etc/apt/sources.list.d/lutris.list
wget -q http://download.opensuse.org/repositories/home:/strycore/xUbuntu_$ver/Release.key -O- | sudo apt-key add -
Instalando dependencias!
Para as dependencias do launcher da Battle.net siga esse guia!!
Sem ele não vai ser possivel instalar o launcher e sem ele não tem como instlar o Starcraft II.
Instalar via Lutris!
A parte mais facil do guia.

Como tudo isso feito nós podemos agora ir para a pagina do Starcraft 2 no Lutris e cliar em instalar, a partir disso é apenas clicar em seguinte e esperar o script terminar de instalar, uma vez isso feito nós podemos abrir o lutris e clicar no icone do Starcraft 2, isso vai abrir o app da Battle.net e instalar o Starcraft 2 normalmente.
